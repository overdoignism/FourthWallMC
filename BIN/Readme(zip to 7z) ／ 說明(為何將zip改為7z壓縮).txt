中文說明：

原本此處提供的格式為 zip 壓縮，但由 windows 解壓縮後仍會將檔案視為下載的，
導致會發生未簽署錯誤。所以從現在起改提供 7z 檔，以確保是從 7zip 解壓縮。

可以從這裡下載 7zip: https://www.7-zip.org/  (免費軟體/開源軟體)

=================================================================================

English:

In the past, here was providing zip file, but if decompress by windows, 
it will not be executable because it's un-signed.

So we provide 7z file to make sure it's decompress by 7zip. 

You can download 7zip here: https://www.7-zip.org/ (freeware/open source)
